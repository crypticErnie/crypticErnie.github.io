Nord Kodi Repository

Add-on repository for the Nord VPN Kodi service.

Install in Kodi

  1. Download the repository zip:  repository.nord-1.0.0.zip
  2. In Kodi, go to Settings ▸ Add‑ons ▸ Install from zip file and select the downloaded file.
  3. After installation, choose Install from repository ▸ Nord Repository ▸ Services ▸ Nord VPN to install the add‑on.

For source code and issue tracking visit the project on GitHub.
